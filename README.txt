Quartz Mod
====================

Adds Quartz ore, block of quartz and relative craft.

Version: 1.1
License: LGPL 2.1 or later
Dependencies: default mod , nether mod, stairs mod 

Report bugs or request help on the forum topic or write me an e-mail.

Installation
------------

Unzip the archive and place it in minetest/mods/

(  GNU/Linux: If you use a system-wide installation place
	it in ~/.minetest/mods/.  )

(  If you only want this to be used in a single world, place
	the folder in worldmods/ in your worlddirectory.  )

For further information or help see:
http://wiki.minetest.com/wiki/Installing_Mods
